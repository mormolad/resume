# Resume — интерактивное портфолио

Сайт‑портфолио с 3D‑каруселью проектов, минималистичными страницами «Обо мне» и «Контакты», а также параласк эффектом на главной.

## Возможности

- 3D‑карусель проектов с автоворотом и паузой по наведению
- Управление каруселью через скролл
- Актуальные карточки портфолио с превью и описанием
- Страницы: `/`, `/portfolio`, `/about_me`, `/contacts`

## Стек

- Next.js 15
- React 19
- TypeScript
- CSS Modules
- styled-components (точечно)

## Быстрый старт

```bash
npm install
npm run dev
```

Открыть: `http://localhost:3000`

## Структура

```
src/
  app/                # страницы и глобальные стили
  components/         # UI-компоненты
  constants/          # данные для портфолио и контактов
public/
  img/                # изображения и превью
  fonts/              # локальные шрифты
```

## Данные

Карточки проектов хранятся в `src/constants/portfolio.ts`.  
Контакты и список тем для связи — в `src/constants/contacts.ts`.

## Шрифты

По умолчанию используется шрифт из `public/fonts`. Подключение — в `src/app/globals.css`.

## Скрипты

- `npm run dev` — локальная разработка
- `npm run build` — сборка
- `npm run start` — запуск production
- `npm run lint` — линтер
